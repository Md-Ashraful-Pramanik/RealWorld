# RealWorld Backend API

## Project Summary

This project implements the RealWorld backend API specification, providing a production-ready RESTful service for articles, users, profiles, tags, comments, and authentication. The goal is a clean, standards-compliant implementation of the endpoints and behaviors defined by the RealWorld spec, suitable for learning, benchmarking, and real-world use.

## Technology Stack

- Node.js backend implementation
- REST API aligned with the RealWorld backend endpoints specification

## Scope

The API will cover all required resources, including:

- User registration, authentication, and profiles
- Article creation, reading, updating, favoriting, and feeds
- Commenting system on articles
- Tag discovery and listing

## Specification

This project follows the RealWorld backend endpoints specification:
https://realworld-docs.netlify.app/specifications/backend/endpoints/
